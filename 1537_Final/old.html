🔢 JavaScript Arrays

Arrays are ordered lists that can hold any data type. Useful methods include:

  • push(): Adds to end
  • pop(): Removes from end
  • shift(): Removes from start
  • unshift(): Adds to start
  • map(), filter(), reduce(): Functional tools
const numbers = [1, 2, 3];
numbers.push(4);            // [1,2,3,4]
const evens = numbers.filter(n => n % 2 === 0);

⚙️ Vanilla JavaScript

Pure JavaScript without frameworks. Key operations:

DOM Selection

  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()
const btn = document.getElementById("submitBtn");
btn.addEventListener("click", () => alert("Clicked!"));

Manipulating DOM

Changing elements dynamically with innerText, innerHTML, and class manipulation.

Events

Attach listeners with addEventListener using event types like click, mouseenter.

🌐 AJAX with Async/Await

AJAX allows async data fetching from APIs using JavaScript:

async function getData() {
  const res = await fetch('/api/users');
  const data = await res.json();
  console.log(data);
}

🚀 Express.js

Minimal and flexible Node.js framework.

  • app.get(), app.post(), app.put(), app.delete()
  • req.params, req.query, req.body
  • res.send(), res.status(), res.json()
app.get('/user/:id', (req, res) => {
  res.send(`User ID: ${req.params.id}`);
});

🗃️ MongoDB

Document-oriented NoSQL database. CRUD operations:

  • Insert: db.collection.insertOne()
  • Find: db.collection.find()
  • Update: db.collection.updateOne()
  • Delete: db.collection.deleteOne()

🧬 Mongoose

Mongoose simplifies MongoDB usage with schemas and models.

const userSchema = new mongoose.Schema({ name: String });
const User = mongoose.model("User", userSchema);

// Async Create
async function createUser() {
  const user = new User({ name: "Kiya" });
  await user.save();
}
